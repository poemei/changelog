# Install
1. Use `install/changelog.sql` to setup the announcements data table
2. Copy all of the contents in `app` to `app` of your Chaos MVC domain.
3. Log into your website and go to its `admin`
4. Click on Changelog
5. Add your first GitHub Like Changelog
6. Navigate to /changelog to view

 - This does depend on the Chaos MVC built-in **Markdown Rendering**
 
 ## Directory
 - Comes with both admin and public views
